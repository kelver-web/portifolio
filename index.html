Projetos

O desenvolvimento de projetos é um processo que envolve a criação de soluções para atender às necessidades de um determinado problema ou objetivo. É uma abordagem sistemática que envolve várias etapas, desde a definição do problema até a entrega da solução final.

Projeto Cadastro de Usuários

O projeto de cadastro de usuários é uma aplicação bem simples que permite registrar informações pessoais de usuários como nome e idade e posteriormente poderem ser deletados, a aplicação tem como objetivo o treinamento em Node.js e o framework React.

Projeto Conversor de Moedas

O projeto Conversor de Moedas é uma aplicação que permite que usuários convertam valores monetários de uma moeda para outra. O objetivo principal é fornecer uma ferramenta simples e fácil de usar para pessoas que precisam converter moedas para fins pessoais ou profissionais. Ver projeto!
